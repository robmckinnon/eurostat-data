# Eurostat data

## Install

The following need to be installed to run processing scripts:

* [csvkit](https://csvkit.readthedocs.io/en/latest/)

To install on a Mac:

```sh
brew install csvkit
```

## Extract and Transform Data

Run make:

```sh
make
```

## Clear Data

Run:

```sh
make clobber
```
